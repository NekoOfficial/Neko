## About this project

This is the official repository for the Neko Discord bot. ~~All changes made to the bot are first updated here then pushed to production code.~~ **Public** changes to the bot will be considered pushing here after 2-3 days of work. Private updates, including music module, will **not** receive updates, in this repo. Removal of data will still happen if eligible.

I made this repo since I keep losing files out of nowhere, so the code wouldn't just die.

If the code is hard to read to you, I can't do anything about it because I really like bodging stuff together to make it work. That's why this code is hard to maintain, but since I don't have much time for that unless requested, I leave it there.

This code works for me, if you have any issues trying to run a modified version or a full copy, I'm not going to assist you in that process.

The database system of this bot was inspired from ravener/miyako, which I felt impressive and clean to use since it is more convenient to update stuff using `mongodb` rather than just using `mongoose`, it sucks. I had nightmares using it.

There's no documentation or whatsoever currently on how to use the commands and stuff but that's for the future. For now I'll let slash commands' description to do its job: to guide people through the commands instead of me making a pointless documentation for it.

## A notice on replicating the code 

You can modify it to suit your needs. But it does not mean that you can discriminate the bot in any way possible.
After modification, please leave this short line at the description (the About Me section, the Server Discovery description and things like that) of your bot:

```Original source code of the bot can be found at https://github.com/NekoOfficial/Neko```
